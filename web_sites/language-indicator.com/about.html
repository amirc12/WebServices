About Language Indicator
Language Indicator is a small yet powerful tool designed for users who work with multiple languages on their computer. Its primary purpose is to display a small tooltip that indicates the current keyboard language before you start typing in any editable field.

Originally, I created this software for myself after repeatedly mistyping text due to using the wrong language. Over time, it evolved into a tool I’m happy to share with others.

I hope you find it as helpful as I do! Your feedback is invaluable, and I’d greatly appreciate any suggestions to improve it in future versions.

Lastly, Language Indicator is completely free for personal use. If you enjoy it, you’re welcome to support the project, perhaps by buying me a coffee (-:

Technically..
Language Indicator is composed of two files. The main file is a DLL that does most of the work and EXE file that simply runs that DLL and also adds the icon in the system tray.

It is written in C++ on Microsoft Visual Studio using native Win32 API calls. Not MFC nor .NET packages are needed
